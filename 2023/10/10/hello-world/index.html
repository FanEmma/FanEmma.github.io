如何建立部落格-001 介紹

1 初衷

在這幾次的面試過程中，幾位面試官有提到的增進技術能力的另一個方法，就是寫部落格．
將所學輸出，測試自己的溝通能力是否有辦法讓別人理解，相對的讓他人有審核自己的機會．
因此建立部落格並撰寫文章．

2 使用到的工具

  • NodeJS
  • Git
  • Hexo
  • Github Page

3 步驟

  1. 安裝環境
  2. 安裝 Hexo
  3. Hexo - 建立文章
  4. Github - 建立 repository
  5. Hexo - 輸出網頁並上傳公開、分享
